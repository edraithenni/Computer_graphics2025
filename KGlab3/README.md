# Сравнительный анализ алгоритмов растеризации отрезков

| Критерий | Пошаговый алгоритм | Алгоритм ЦДА | Алгоритм Брезенхема |
|----------|---------------------|--------------|---------------------|
| **Математическая основа** | y = kx + b с округлением | Δy/Δx, дифференциальное уравнение | Целочисленная ошибка между линией и пикселями |
| **Тип связности** | 8-связная | 8-связная | 8-связная |
| **Арифметика** | Вещественные числа (умножение, округление) | Вещественные числа (деление на каждом шаге) | Только целые числа (сложение, вычитание, сдвиг) |
| **Операции на шаг** | 1 умножение + 1 округление | 1 деление + 1 сложение | 2-3 целочисленные операции |
| **Качество линии** | Среднее (погрешности округления) | Среднее (зависит от метода) | Высокое (минимальная ошибка) |
| **Скорость** | Низкая | Средняя | Высокая |
| **Память** | 2 вещественных переменных | 2 вещественных переменных | 3-4 целочисленных переменных |
| **Деление** | Нет | Да (на каждом шаге) | Нет (в целочисленной версии) |
| **Округление** | Да (на каждом шаге) | Да/Нет (зависит от реализации) | Нет |
| **Поддержка октантов** | Требует модификации | Есть версии для всех октантов | Сводится к 1 октанту через симметрию |
| **Практическое применение** | Учебные цели | Исторический интерес, не применяется | Стандарт в графических системах |
| **Год создания** | - | - | 1962 |
| **Автор** | - | - | Джек Е. Брезенхем (IBM) |
| **Особенности** | Самый простой для понимания | Основан на дифф. уравнениях | Имеет модификации для окружностей, эллипсов |
